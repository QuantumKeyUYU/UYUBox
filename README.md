# UYUBox Quantum

UYUBox Quantum — это мобильный клиент следующего поколения для защищённой работы
с контейнерами `zilant-prime-core`. Приложение превращает локальный офлайн-шифратор
в zero-trust платформу, готовую к корпоративному использованию, и служит
витриной для инновационных возможностей, описанных в нашей стратегии ×10000.

## Новые ключевые возможности

- **Адаптивные криптопрофили.** Встроенный `CryptoManager` анализирует состояние
  устройства и предлагает оптимальный Argon2id-профиль (light/balanced/hardened/ultra).
- **Модульные пайплайны.** Экран «Пайплайны» даёт визуальное представление о потоках
  «сканировать → классифицировать → шифровать → доставлять» и позволяет запускать
  заранее подготовленные рецепты. Логика вынесена в `app/services/workflows.py`.
- **Secure Gateway.** Экран «Zero-Trust Gateway» моделирует P2P-сессии с
  постквантовыми наборами (Kyber + Dilithium) и управлением участниками. Функционал
  реализован в `app/services/secure_share.py`.
- **Прозрачный аудит.** Каждое действие пользователя (упаковка, распаковка,
  создание сессии) логируется в Merkle-подобный реестр (`app/services/audit.py`),
  а результаты отображаются на экране «Аудит».
- **Обновлённый UI.** Приложение использует KivyMD с навигацией между домашним
  экраном, пайплайнами, шлюзом, аудитом и файловым выборщиком. Главная сцена
  отображает телеметрию устройства, рекомендации по профилю безопасности,
  позволяет управлять метаданными `.zilx`-контейнеров и готова к интеграции AI.


## Быстрый старт для разработчика

1. **Установите зависимости** (Python 3.10+, Cython, Kivy, KivyMD). Проще всего —
   `python -m pip install -r requirements.txt` внутри виртуального окружения.
2. **Запустите приложение на десктопе** для предпросмотра:
   ```bash
   python main.py
   ```
   При первом запуске незагруженные модули ядра будут заменены безопасными
   заглушками, чтобы интерфейс продолжил работать.
3. **Сборка APK** выполняется через Buildozer: `buildozer -v android debug`. Первый
   прогон скачает Android SDK/NDK и может занять до часа.

## Архитектурные акценты

- `CryptoManager` инкапсулирует операции `pack_file`/`unpack_file`/`read_metadata`
  и собирает отчёты для UI, что позволяет позднее внедрить двойное шифрование,
  PQC-обёртки и биометрию без переписывания экранов.
- `SecureShareGateway` управляет сессиями zero-trust обмена и фиксирует события в
  аудит-реестре. Переход на реальные WebRTC/SignalR транспортные уровни предполагает
  замену лишь сервиса, не UI.
- `AuditLedger` строит цепочки хэшей и подготавливает данные для экспорта в внешние
  реестры (Hyperledger, zk-proof сервисы).
- `WorkflowsScreen` отображает рецепты, которые впоследствии можно собрать через
  drag-and-drop редактор и хранить в формате `.zilx` вместе с политиками доступа.

## Дальнейшие шаги к ×10000

- **Интеграция PQC через `oqs-python`** и аппаратные токены (StrongBox, FIDO2).
- **Runtime-integrity мониторинг**: root/emulator detection, Play Integrity, decoy UI.
- **Marketplace политик** на основе sandboxed WASM-надстроек.
- **Экспорт Zero-Knowledge доказательств** владения/неизменности контейнера.
